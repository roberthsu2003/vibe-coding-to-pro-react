# 00 - HTML + Tailwind CSS 入門

## 主題說明

**Tailwind CSS** 是 **utility-first** 的樣式框架：你在 HTML 上直接加上許多小型 class（例如 `p-4`、`text-center`、`bg-slate-100`），由框架產生對應 CSS，不必先寫一大份自訂樣式表。本單元假設你已讀過 [00 - HTML 與 CSS](../00-HTML-CSS/README.md)（或具同等基礎），並以 **官方 Play CDN** 在單一 HTML 檔內體驗 Tailwind——**不涉及** npm、PostCSS 或正式建置流程（那些會在 [02 - Vite 入門](../02-Vite入門/主題.md) 等章節出現）。

### 學習目標

完成本單元後，你將能夠：

- ✅ 說明 utility-first 與「在 HTML 上堆疊 class」的開發感受
- ✅ 使用 Play CDN 載入 Tailwind，並辨識幾組常用 utility（間距、字級、顏色、Flex 簡單排版）
- ✅ 知道 CDN 方式適合學習與原型，正式專案通常改走建置流程
