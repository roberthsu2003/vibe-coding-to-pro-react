# 範例 0：建立進階練習專案

## 目標

在 `09-React/examples` 建立 **Vite + React + TypeScript** 專案，作為本單元範例 1～7 的練習環境。

---

## 前置條件

- 已完成 [05 - React 核心概念](../05-React核心概念/主題.md) 範例 0～5  
- 已熟悉 [00 - Node.js 與 npm](../00-Node.js與npm/README.md) 與 [01 - 環境建置](../01-環境建置/主題.md) 的基本指令  

---

## 步驟 1：建立專案

在 `09-React` 資料夾中執行：

```bash
cd 09-React
npm create vite@latest examples -- --template react-ts
```

> **若已存在 `examples` 資料夾**：可先重新命名備份，或刪除後再執行。

---

## 步驟 2：安裝依賴並啟動

```bash
cd examples
npm install
npm run dev
```

瀏覽器開啟終端機顯示的本機網址（通常為 `http://localhost:5173`）。

---

## 步驟 3：建議的檔案結構

後續範例會請你在 `src/` 新增元件，可採用：

```
src/
├── App.tsx
├── main.tsx
├── components/        ← 可放各範例元件
│   ├── RefDemo.tsx
│   ├── MemoDemo.tsx
│   └── ...
└── hooks/             ← 自訂 Hook（範例 4）
    └── useToggle.ts
```

依各範例說明，將 `App.tsx` 改為匯入對應示範元件即可。

---

專案就緒後，請繼續 [範例 1：useRef 與 DOM](範例1-useRef與DOM.md)。
