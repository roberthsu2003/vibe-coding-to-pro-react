# 06 - Remix 與 React Router v7

## 主題說明

本單元介紹 **Remix** 與 **React Router v7** 的關係，以及如何用 React Router v7（整合 Remix 的全端框架）建立現代化 Web 應用。對象為已學過 React 核心概念的初學者。

### 學習目標

- 理解 React 與 Remix、React Router v7 的關係
- 學會建立 React Router v7 專案
- 認識專案結構與路由配置
- 掌握 Loader 載入資料
- 學會 Action 處理表單
