# 範例 2：認識專案結構

## 目標

認識 React Router v7 專案的資料夾與檔案結構。

---

## 前置條件

- 已完成 [範例 0：建立專案](範例0-建立專案.md)

---

## 步驟 1：專案根目錄

```
examples/
├── app/              ← 應用程式原始碼
├── public/           ← 靜態資源（favicon 等）
├── package.json
├── react-router.config.ts   ← React Router 設定
├── vite.config.ts
└── tsconfig.json
```

---

## 步驟 2：app 資料夾（建立專案後的初始結構）

完成範例 0 後，`create-react-router` 預設會產生：

```
app/
├── root.tsx         ← 根佈局（HTML、Meta、Outlet）
├── app.css          ← 全域樣式
├── routes.ts        ← 路由設定
├── routes/          ← 路由頁面
│   └── home.tsx     ← 首頁
└── welcome/         ← 共用元件
    └── welcome.tsx
```

> **說明**：`about`、`products`、`contact` 及 `examples/` 會在範例 3～5 中逐步加入。

---

## 步驟 3：重要檔案說明

| 檔案 | 說明 |
|------|------|
| **root.tsx** | 根元件，定義 HTML 結構、`<Outlet />` 渲染子路由 |
| **routes.ts** | 定義 URL 與頁面的對應 |
| **routes/home.tsx** | 首頁（`/`）的頁面元件 |

---

## 步驟 4：查看 routes.ts

建立專案後的 `app/routes.ts` 預設為：

```ts
import { type RouteConfig, index } from "@react-router/dev/routes";

export default [index("routes/home.tsx")] satisfies RouteConfig;
```

- `index("routes/home.tsx")`：首頁 `/` 對應 `home.tsx`
- 之後在範例 3～5 會依序加入 `route("about", ...)`、`route("products", ...)`、`route("contact", ...)`

> **說明**：本章節的 `examples` 資料夾為**最原始的基本 template**，請依序完成範例 3～5，依照教學修改檔案，逐步加入 About、Products、Contact 頁面。

---
