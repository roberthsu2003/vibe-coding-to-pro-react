# 範例 2：認識專案結構

## 目標

認識 React Router v7 專案的資料夾與檔案結構。

---

## 前置條件

- 已完成 [範例 0：建立專案](範例0-建立專案.md)

---

## 步驟 1：專案根目錄

```
examples/
├── app/              ← 應用程式原始碼
├── public/           ← 靜態資源（favicon 等）
├── package.json
├── react-router.config.ts   ← React Router 設定
├── vite.config.ts
└── tsconfig.json
```

---

## 步驟 2：app 資料夾

```
app/
├── root.tsx         ← 根佈局（HTML、Meta、Outlet）
├── app.css          ← 全域樣式
├── routes.ts        ← 路由設定
├── routes/          ← 路由頁面
│   ├── home.tsx     ← 首頁（含 React 核心概念範例）
│   ├── about.tsx    ← 關於頁
│   ├── products.tsx ← 商品頁（Loader 範例）
│   └── contact.tsx  ← 聯絡頁（Action 範例）
├── examples/        ← React 核心概念範例（與 05 章節一致）
│   ├── Example1JSX.tsx
│   ├── Example2Props.tsx
│   ├── Example3State.tsx
│   ├── Example4UseEffect.tsx
│   └── Example5Events.tsx
└── welcome/         ← 共用元件（選用）
    └── welcome.tsx
```

---

## 步驟 3：重要檔案說明

| 檔案 | 說明 |
|------|------|
| **root.tsx** | 根元件，定義 HTML 結構、`<Outlet />` 渲染子路由 |
| **routes.ts** | 定義 URL 與頁面的對應 |
| **routes/home.tsx** | 首頁（`/`），整合 05 章節的 React 核心概念互動範例 |
| **examples/** | React 核心概念範例元件（JSX、Props、State、useEffect、事件處理） |

---

## 步驟 4：查看 routes.ts

開啟 `app/routes.ts`：

```ts
import { type RouteConfig, index, route } from "@react-router/dev/routes";

export default [
  index("routes/home.tsx"),
  route("about", "routes/about.tsx"),
  route("products", "routes/products.tsx"),
  route("contact", "routes/contact.tsx"),
] satisfies RouteConfig;
```

- `index("routes/home.tsx")`：首頁 `/` 對應 `home.tsx`，顯示 React 核心概念互動範例
- `route("about", ...)`：`/about` 關於頁
- `route("products", ...)`：`/products` 商品頁（Loader 範例）
- `route("contact", ...)`：`/contact` 聯絡頁（Action 範例）
