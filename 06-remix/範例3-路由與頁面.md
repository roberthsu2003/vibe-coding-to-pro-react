# 範例 3：路由與頁面

## 目標

學會新增路由與頁面，理解 `route`、`index`、`Outlet` 的用法。

---

## 前置條件

- 已完成 [範例 2：認識專案結構](範例2-認識專案結構.md)

---

## 步驟 1：認識路由 API

在 `routes.ts` 中常用：

| API | 說明 | 範例 |
|-----|------|------|
| `index()` | 首頁或巢狀預設頁 | `index("routes/home.tsx")` |
| `route()` | 一般路由 | `route("about", "routes/about.tsx")` |
| `layout()` | 共用佈局 | `layout("routes/layout.tsx", [...])` |

---

## 步驟 2：新增 About 頁面

在 `app/routes/` 下建立 `about.tsx`：

```tsx
export default function About() {
  return (
    <main className="p-8">
      <h1 className="text-2xl font-bold">關於我們</h1>
      <p>這是使用 React Router v7 建立的頁面。</p>
    </main>
  );
}
```

---

## 步驟 3：註冊路由

修改 `app/routes.ts`：

```ts
import { type RouteConfig, index, route } from "@react-router/dev/routes";

export default [
  index("routes/home.tsx"),
  route("about", "routes/about.tsx"),
] satisfies RouteConfig;
```

---

## 步驟 4：加入導覽連結

在 `root.tsx` 的導覽列中已有各頁連結；若要在 `home.tsx` 內加入 `<Link>`，可寫：

```tsx
import { Link } from "react-router";

// 在 JSX 中
<Link to="/about">關於我們</Link>
```

> **提示**：本專案的 `root.tsx` 已包含首頁、關於、商品、聯絡的導覽連結。

---

## 步驟 5：驗證

1. 儲存檔案
2. 開啟 `http://localhost:5173`
3. 點擊「關於我們」連結，應導向 `/about`
