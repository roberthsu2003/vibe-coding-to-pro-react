# 範例 3：路由與頁面

## 目標

學會新增路由與頁面，理解 `route`、`index`、`Outlet` 的用法。

---

## 前置條件

- 已完成 [範例 2：認識專案結構](範例2-認識專案結構.md)

---

## 步驟 1：認識路由 API

在 `routes.ts` 中常用：

| API | 說明 | 範例 |
|-----|------|------|
| `index()` | 首頁或巢狀預設頁 | `index("routes/home.tsx")` |
| `route()` | 一般路由 | `route("about", "routes/about.tsx")` |
| `layout()` | 共用佈局 | `layout("routes/layout.tsx", [...])` |

---

## 步驟 2：新增 About 頁面

在 `app/routes/` 下建立 `about.tsx`：

```tsx
export default function About() {
  return (
    <main className="p-8">
      <h1 className="text-2xl font-bold">關於我們</h1>
      <p>這是使用 React Router v7 建立的頁面。</p>
    </main>
  );
}
```

---

## 步驟 3：註冊路由

修改 `app/routes.ts`：

```ts
import { type RouteConfig, index, route } from "@react-router/dev/routes";

export default [
  index("routes/home.tsx"),
  route("about", "routes/about.tsx"),
] satisfies RouteConfig;
```

---

## 步驟 4：在 root.tsx 加入導覽列

開啟 `app/root.tsx`，在 `App` 元件中加入導覽列，讓使用者可以切換頁面：

1. 在 import 中加入 `Link`：
   ```tsx
   import { Link, Outlet, ... } from "react-router";
   ```

2. 將 `return <Outlet />` 改為：
   ```tsx
   return (
     <>
       <nav className="flex gap-4 p-4 border-b">
         <Link to="/" className="text-blue-600 hover:underline">首頁</Link>
         <Link to="/about" className="text-blue-600 hover:underline">關於我們</Link>
       </nav>
       <Outlet />
     </>
   );
   ```

> **說明**：之後範例 4、5 會再於導覽列新增商品、聯絡的連結。

---

## 步驟 5：驗證

1. 儲存檔案
2. 開啟 `http://localhost:5173`
3. 點擊「關於我們」連結，應導向 `/about`
