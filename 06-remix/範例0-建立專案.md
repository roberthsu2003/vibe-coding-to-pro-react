# 範例 0：建立專案

## 目標

使用官方 CLI 建立 React Router v7 專案，作為後續範例的基礎。

---

## 步驟 1：建立專案

在 `06-remix` 資料夾中開啟終端機，執行：

```bash
cd 06-remix
npx create-react-router@latest examples
```

> **若已存在 examples 資料夾**：可先重新命名為 `examples-backup`，或刪除後再執行上述指令。

---

## 步驟 2：選擇選項（若為互動模式）

若 CLI 詢問選項，可依序選擇：

- **Project directory**：使用 `examples`（或直接按 Enter）
- **Template**：使用預設（default）

或加上 `--yes` 使用預設值：

```bash
npx create-react-router@latest examples --yes
```

---

## 步驟 3：進入專案並安裝依賴

若 CLI 未自動安裝，請執行：

```bash
cd examples
npm install
```

---

## 步驟 4：啟動開發伺服器

```bash
npm run dev
```

瀏覽器開啟 `http://localhost:5173` 後，應看到首頁。本專案範例已整合 [05 - React 核心概念](../05-React核心概念/主題.md) 的互動範例，首頁可切換 JSX、Props、State、useEffect、事件處理等範例。
