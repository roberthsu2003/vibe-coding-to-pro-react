# 01 - 環境建置

## 主題說明

本單元學習如何建立 **React + TypeScript + Vite** 的開發環境，以及專案的基本結構。Vite 與 React 預設使用 **ES Module**（`import`／`export`），與前置單元 **00** 的寫法一致。

### 學習目標

- 了解前置需求（Node.js、npm）
- 認識 Vite 專案的目錄結構
- 掌握常用指令（dev、build、preview）
- 理解從啟動到畫面的流程
