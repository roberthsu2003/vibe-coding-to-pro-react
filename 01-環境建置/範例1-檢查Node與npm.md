# 範例 1：檢查 Node 與 npm

## 目標

確認電腦已安裝 Node.js 和 npm，這是建立 React 專案的前置需求。

---

## 步驟 1：開啟終端機

- **Windows**：按 `Win + R`，輸入 `cmd` 或使用 PowerShell
- **Mac**：按 `Cmd + 空格`，輸入「終端機」
- **VS Code**：按 `` Ctrl + ` `` 開啟內建終端機

---

## 步驟 2：檢查 Node.js 版本

在終端機輸入：

```bash
node -v
```

**預期結果**：顯示版本號，例如 `v20.10.0`

- 若顯示「找不到指令」，請安裝 Node.js：https://nodejs.org/
- 建議使用 v18 以上版本

---

## 步驟 3：檢查 npm 版本

在終端機輸入：

```bash
npm -v
```

**預期結果**：顯示版本號，例如 `10.2.0`

- npm 會隨 Node.js 一併安裝，通常不需額外安裝

---

## 完成

若兩個指令都能顯示版本號，代表環境已就緒，可以進行下一章範例。
