# 範例 3：認識專案結構

## 目標

了解 Vite + React 專案中各檔案與資料夾的作用。

---

## 步驟 1：查看專案根目錄

建立專案後，根目錄大致如下：

```
my-react-app/
├── index.html
├── package.json
├── vite.config.ts
├── tsconfig.json
├── tsconfig.node.json
├── public/
└── src/
```

---

## 步驟 2：認識各檔案

| 檔案／資料夾 | 說明 |
|-------------|------|
| **index.html** | 網頁入口，內有 `<div id="root">`，React 會把畫面渲染到這裡 |
| **package.json** | 專案設定：名稱、版本、依賴套件、腳本指令 |
| **vite.config.ts** | Vite 的設定檔 |
| **tsconfig.json** | TypeScript 的設定檔 |
| **public/** | 靜態資源（圖片、favicon 等），不經過打包 |
| **src/** | 主要程式碼（React 元件、樣式等） |

---

## 步驟 3：查看 src 資料夾

```
src/
├── main.tsx      # 程式入口，掛載 React 到 #root
├── App.tsx       # 主元件
├── App.css       # App 的樣式
├── index.css     # 全域樣式
└── vite-env.d.ts # Vite 相關型別宣告
```

**重點**：`main.tsx` 會找到 `index.html` 中的 `#root`，並把 React 應用掛載上去。

---
