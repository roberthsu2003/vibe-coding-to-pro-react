# 範例 4：啟動開發伺服器

## 目標

啟動 Vite 開發伺服器，在瀏覽器看到 React 應用畫面。

---

## 步驟 1：確認在專案目錄

```bash
cd my-react-app
```

---

## 步驟 2：執行啟動指令

```bash
npm run dev
```

---

## 步驟 3：查看終端機輸出

終端機會顯示類似：

```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
```

---

## 步驟 4：開啟瀏覽器

在瀏覽器網址列輸入：`http://localhost:5173/`

**預期結果**：看到 Vite + React 的歡迎畫面。

---

## 步驟 5：體驗熱更新

1. 用編輯器開啟 `src/App.tsx`
2. 修改其中的文字（例如把 "Vite" 改成 "我的專案"）
3. 儲存檔案
4. 觀察瀏覽器畫面是否自動更新（無需手動重新整理）

---

## 流程說明

```
npm run dev
    ↓
Vite 啟動開發伺服器
    ↓
瀏覽器載入 index.html
    ↓
載入 main.tsx → 掛載 React → 渲染 App.tsx
    ↓
顯示畫面
```

---

## 完成

你已成功啟動開發環境。其他常用指令：

- `npm run build`：建置生產版本
- `npm run preview`：預覽建置結果（需先執行 build）
