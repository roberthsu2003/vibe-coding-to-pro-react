# 00 - HTML 與 CSS 入門

## 主題說明

本單元只談**標記（HTML）**與**樣式（CSS）**：瀏覽器如何顯示文字、連結、清單與版面。不涉及 JavaScript、模組或建置工具；若你已熟悉網頁骨架與基本樣式，可進入 [00 - HTML + Tailwind CSS](../00-HTML-TailwindCSS/README.md) 或略過並直接讀 [00 - HTML + CSS + JavaScript](../00-HTML-CSS-JavaScript/README.md)。

### 學習目標

完成本單元後，你將能夠：

- ✅ 說明一份 HTML 文件的基本結構（`<!DOCTYPE>`、`html`、`head`、`body`）
- ✅ 使用常見標籤表達語意（標題、段落、連結、清單等）
- ✅ 以 CSS 選擇器與屬性調整顏色、字體、間距，並理解「盒模型」在做什麼
