# 範例 1：網頁骨架與樣式入門

## 目標

用一個資料夾、兩個檔案（`index.html`、`styles.css`）做出可在瀏覽器直接開啟的靜態頁面，並練習「HTML 負責結構、CSS 負責外觀」的分工。

## 步驟 1：建立 `index.html`

下列為**最小且語意清楚**的起手式：`lang` 有助無障礙與搜尋；`meta viewport` 讓手機瀏覽比例正確；`<title>` 會顯示在分頁標題。

```html
<!DOCTYPE html>
<html lang="zh-Hant">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>我的第一頁</title>
    <link rel="stylesheet" href="styles.css" />
  </head>
  <body>
    <header>
      <h1>歡迎</h1>
      <p>這是一段簡單介紹。</p>
    </header>
    <main>
      <h2>待辦</h2>
      <ul>
        <li>閱讀 HTML 結構</li>
        <li>試著改 CSS 顏色</li>
      </ul>
      <p>
        更多資源請見
        <a href="https://developer.mozilla.org/zh-TW/">MDN</a>。
      </p>
    </main>
  </body>
</html>
```

**可對照記的幾個標籤**：`<h1>`～`<h6>` 標題層級、`<p>` 段落、`<ul>`／`<li>` 無序清單、`<a href="…">` 連結。`<header>`、`<main>` 屬於**語意標籤**，讓結構一目了然（也利於無障礙與樣式選取）。

## 步驟 2：建立 `styles.css`

CSS 透過**選擇器**指定「要套用在誰身上」，再用**屬性**設定外觀。

```css
/* 元素選擇器：所有 body 內預設字級與行高 */
body {
  font-family: system-ui, sans-serif;
  line-height: 1.5;
  margin: 0;
  padding: 1rem;
  background-color: #f5f5f5;
  color: #222;
}

/* 後代選擇器：main 裡的 h2 */
main h2 {
  margin-top: 1.5rem;
  color: #1a365d;
}

/* 連結預設有底線，可微調 */
a {
  color: #2b6cb0;
}

a:hover {
  text-decoration: underline;
}
```

**盒模型（點到為止）**：多數區塊元素可想像成「盒子」——`margin` 是與外側的距離，`padding` 是內容與邊框內側的距離；寬高與 `border` 也會影響版面。先習慣改 `margin`／`padding`／`color`／`background-color`，之後可讀 [00 - HTML + Tailwind CSS](../00-HTML-TailwindCSS/README.md) 體驗另一種寫樣式的方式，再進入 [00 - HTML + CSS + JavaScript](../00-HTML-CSS-JavaScript/README.md) 與版面、腳本一起運用。

## 步驟 3：在瀏覽器開啟

將 `index.html` 用瀏覽器開啟（雙擊檔案或拖曳到視窗）。修改 HTML 或 CSS 後**重新整理**即可看到變化。

## 小結

- HTML：描述「有什麼、層級與語意為何」。
- CSS：描述「長什麼樣子、間距與色彩」。
- 下一章可先讀 **Tailwind（CDN）** 或略過，再在同一套基礎上加入 **JavaScript** 與 **ES Module**。
