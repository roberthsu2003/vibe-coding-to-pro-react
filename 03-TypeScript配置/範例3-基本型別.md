# 範例 3：TypeScript 基本型別

## 目標

學會宣告變數的型別，讓程式更安全、更好維護。

---

## 步驟 1：字串型別

```typescript
const name: string = '小明'
```

- `string` 表示此變數只能放文字

---

## 步驟 2：數字型別

```typescript
const age: number = 18
```

- `number` 表示此變數只能放數字

---

## 步驟 3：布林型別

```typescript
const isStudent: boolean = true
```

- `boolean` 表示 true 或 false

---

## 步驟 4：陣列型別

```typescript
const hobbies: string[] = ['閱讀', '運動']
```

- `string[]` 表示「字串的陣列」

---

## 步驟 5：物件型別

```typescript
const user: { name: string; age: number } = {
  name: '小明',
  age: 18,
}
```

- 物件可指定每個屬性的型別

---

## 步驟 6：可選屬性

```typescript
interface Person {
  name: string
  age?: number  // ? 表示 age 可以不存在
}

const p1: Person = { name: '小明' }           // OK
const p2: Person = { name: '小華', age: 20 }  // OK
```

---

## 完成

掌握基本型別後，在 React 元件中定義 Props 時會經常用到。
