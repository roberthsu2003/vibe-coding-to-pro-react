## 步驟 1：函式宣告與參數型別

在參數名稱後加上 `: 型別`，約定傳入的資料形態：

```typescript
function greet(name: string): string {
  return `你好，${name}`
}

console.log(greet('小明'))
```

- 參數 `name` 必須是 `string`  
- 冒號後面的第二個型別（此例為 `string`）是**回傳值型別**

若呼叫時傳錯型別，TypeScript 會報錯：

```typescript
// greet(123)  // ❌ number 不可傳給 string
```

---

## 步驟 2：回傳值型別與推斷

**建議**為公開 API、較長的函式寫明回傳型別，方便閱讀與重構：

```typescript
function add(a: number, b: number): number {
  return a + b
}
```

若函式本體直接 `return` 字面表達式，TypeScript 常能**推斷**回傳型別，此時可省略 `: number`（團隊規範允許時）：

```typescript
const double = (n: number) => n * 2  // 推斷回傳為 number
```

**沒有 `return` 或只執行副作用**時，回傳型別為 **`void`**（見步驟 5）。

---

## 步驟 3：箭頭函式

箭頭函式與一般函式在型別標註上概念相同：

```typescript
const multiply = (a: number, b: number): number => a * b

const sayHi = (name: string) => {
  console.log(`Hi, ${name}`)
}
```

多行區塊時若需要回傳值，仍須使用 `return`：

```typescript
const clamp = (value: number, min: number, max: number): number => {
  if (value < min) return min
  if (value > max) return max
  return value
}
```

---

## 步驟 4：選用參數（`?`）

在參數後加 `?` 表示**可以省略**，此時型別會自動與 `undefined` 聯合（呼叫端可不傳）：

```typescript
function introduce(name: string, age?: number): string {
  if (age !== undefined) {
    return `${name}，${age} 歲`
  }
  return name
}

introduce('小明')       // OK
introduce('小華', 20)   // OK
```

> **注意**：選用參數必須放在**必要參數後面**。

---

## 步驟 5：預設參數與 `void`

**預設值**可讓參數在省略時使用預設：

```typescript
function createLabel(text: string, prefix = '標籤'): string {
  return `${prefix}：${text}`
}

createLabel('重要')              // 「標籤：重要」
createLabel('警告', '注意')       // 「注意：警告」
```

執行動作但**不回傳有意義的值**時，標註為 **`void`**：

```typescript
function logScore(score: number): void {
  console.log(`分數：${score}`)
}
```

---

## 步驟 6：`never`（選讀）

若函式**永遠不會正常結束**（例如一定拋錯，或無窮迴圈），回傳型別為 **`never`**：

```typescript
function fail(message: string): never {
  throw new Error(message)
}
```

一般應用較少手寫，先知道即可。

## 重點整理

| 概念 | 說明 |
|------|------|
| `(a: number, b: number)` | 參數型別 |
| `: string`（函式後） | 回傳值型別 |
| `age?` | 選用參數，可省略 |
| `prefix = '標籤'` | 預設參數 |
| `: void` | 沒有回傳值（或忽略回傳值） |
| 箭頭函式 | `(x: number): number => ...` |

## 小練習

在 **tsx-cli** 或 **網頁專案** 的練習檔中完成：

1. 寫一個函式 `isPassing(score: number): boolean`，當 `score >= 60` 回傳 `true`，否則 `false`。  
2. 寫一個函式 `repeat(text: string, times?: number)`：若未傳 `times`，預設重複 `1` 次（回傳字串）。提示：可併用 `?` 與預設值，或只在參數使用預設值。
