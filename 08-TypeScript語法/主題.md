# 08 - TypeScript 語法

## 主題說明

本單元專門講解 **TypeScript 語法**：變數與型別註解、函式、介面、泛型、聯合型別等，讓你在閱讀與撰寫 React / Vite 專案時能對型別系統有完整概念。

對象為已讀過 [03 - TypeScript 配置](../03-TypeScript配置/主題.md)（tsconfig、`.ts` / `.tsx`）的初學者；本單元**不**重複專案設定，聚焦在**程式語法**。

### 學習目標

- 熟練使用型別註解與基本型別
- 為函式參數與回傳值標註型別
- 使用 `interface` 與 `type` 描述物件與結構
- 認識陣列、泛型與常見集合型別
- 理解聯合型別、型別窄化與實務常用寫法
