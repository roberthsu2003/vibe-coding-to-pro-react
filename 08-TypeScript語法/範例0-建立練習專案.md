# 範例 0：建立練習環境（指令列 → 網頁）

## 目標

準備兩種練習方式，並建議**先簡後繁**：

1. **階段一（指令列）**：用 **tsx** 在終端機執行 `.ts`，最快看到 `console.log` 結果。  
2. **階段二（網頁）**：用 **Vite** 在瀏覽器執行，與實際前端專案較接近，並可同時看 **Console** 與頁面輸出。

> **名稱說明**：有人口頭會說「**tcx**」這類寫法，一般是指「在終端機直接跑 TypeScript」；本教材使用的工具套件名稱是 **`tsx`**（請勿與 React 元件副檔名 `.tsx` 混淆）。

## 前置條件

- 已完成 [01 - 環境建置](../01-環境建置/主題.md) 與 [03 - TypeScript 配置](../03-TypeScript配置/主題.md) 的基礎認識

## 階段一：指令列練習（建議先做）

請使用 **`08-TypeScript語法/tsx-cli/`**：

- 內含與範例 1～5 對應的 **`src/01-basics.ts`**～**`src/05-unions.ts`**。
- 安裝後以 `npm run 01`～`npm run 05`（或 `npx tsx src/檔名.ts`）執行，輸出在**終端機**。

詳見 [**tsx-cli 說明**](tsx-cli/README.md)。

```bash
cd 08-TypeScript語法/tsx-cli
npm install
npm run 01
```

---

## 階段二：瀏覽器練習（銜接前端）

完成階段一、或希望直接體驗網頁時，再使用 **`08-TypeScript語法/examples/`** 下各範例資料夾：

| 範例 | 資料夾 |
|------|--------|
| 範例 1 | `examples/01-basics/` |
| 範例 2 | `examples/02-functions/` |
| 範例 3 | `examples/03-interfaces/` |
| 範例 4 | `examples/04-arrays-generics/` |
| 範例 5 | `examples/05-unions/` |

每個資料夾內含 **`index.html`**、**`src/main.ts`**、**`src/style.css`**、該範例的 **`src/practice-….ts`** 與 Vite 設定。將資料夾複製到任意位置後：

```bash
cd 該範例資料夾
npm install
npm run dev
```

詳見 [examples 目錄說明](examples/README.md)。

---

## 替代做法：自行建立空白 Vite 專案

若你希望從零建立（例如自訂資料夾名稱），可在 **`08-TypeScript語法`** 下執行：

```bash
cd 08-TypeScript語法
npm create vite@latest my-ts-practice -- --template vanilla-ts
cd my-ts-practice
npm install
npm run dev
```

再自行新增練習用 `.ts` 並在 `main.ts` 以 `import` 載入；若要與講義一致，可對照各 `examples/` 資料夾內的 `main.ts`（含將 `console.log` 同步到頁面的寫法）。
