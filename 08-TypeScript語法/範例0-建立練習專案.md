# 範例 0：建立練習環境（指令列 → 網頁）

## 目標

準備兩種練習方式，並建議**先簡後繁**：

1. **階段一（指令列）**：用 **tsx** 在終端機執行 `.ts`，最快看到 `console.log` 結果。  
2. **階段二（網頁）**：用 **Vite** 在瀏覽器執行，以**獨立設計的畫面**呼應各範例的 TypeScript 觀念（與 tsx-cli **不必**逐行相同）；除錯時仍可使用 **Console**。

> **名稱說明**：本教材使用的工具套件名稱是 **`tsx`**（請勿與 React 元件副檔名 `.tsx` 混淆）。

### 練習原則（與範例 1～6 相同）

本教材附帶的 **`tsx-cli/src/*.ts`**、**`examples/*/src/practice-*.ts`** 若已含完整程式，僅供**對照答案**。**請在你自己的練習檔**中，依各範例講義**手動輸入**或**從講義複製**程式後再執行；**不要**以「只執行 repo 裡現成檔案、自己不寫」當作完成。

## 階段一：指令列練習（建議先做）

請使用 **`08-TypeScript語法/tsx-cli/`**：

- 內含與範例 1～6 對應的 **`src/01-basics.ts`**～**`src/06-utility-types.ts`**（可作為完成後的**參考解答**）。
- 環境建置好後，請在**自己的練習檔**撰寫程式，再以 `npm run 01`～`npm run 06`（或 `npx tsx src/檔名.ts`）執行**你的檔案**，輸出在**終端機**。

詳見 [**tsx-cli 說明**](tsx-cli/README.md)。

```bash
cd 08-TypeScript語法/tsx-cli
npm install
npm run 01
```

## 階段二：瀏覽器練習（銜接前端）

完成階段一、或希望直接體驗網頁時，再使用 **`08-TypeScript語法/examples/`** 下各範例資料夾：

| 範例 | 資料夾 |
|------|--------|
| 範例 1 | `examples/01-basics/` |
| 範例 2 | `examples/02-functions/` |
| 範例 3 | `examples/03-interfaces/` |
| 範例 4 | `examples/04-arrays-generics/` |
| 範例 5 | `examples/05-unions/` |

每個資料夾內含 **`index.html`**、**`src/main.ts`**、**`src/style.css`**、該範例的 **`src/practice-….ts`** 與 Vite 設定。將資料夾複製到任意位置後：

```bash
cd 該範例資料夾
npm install
npm run dev
```

詳見 [examples 目錄說明](examples/README.md)。

## 替代做法：自行建立空白 Vite 專案

若你希望從零建立（例如自訂資料夾名稱），可在 **`08-TypeScript語法`** 下執行：

```bash
cd 08-TypeScript語法
npm create vite@latest my-ts-practice -- --template vanilla-ts
cd my-ts-practice
npm install
npm run dev
```

再自行新增練習用 `.ts` 並在 `main.ts` 以 `import` 載入；可對照各 `examples/` 資料夾內的 `main.ts`（僅載入樣式與 `practice-….ts`）與 `practice-….ts`（畫面與型別邏輯）。
