# 範例 0：建立練習專案（可選）

## 目標

在 `08-TypeScript語法/examples` 建立一個 **Vite + TypeScript** 空專案，作為後續語法範例的練習環境。

---

## 前置條件

- 已完成 [01 - 環境建置](../01-環境建置/主題.md) 與 [03 - TypeScript 配置](../03-TypeScript配置/主題.md) 的基礎認識

---

## 步驟 1：建立專案

在 `08-TypeScript語法` 資料夾中執行：

```bash
cd 08-TypeScript語法
npm create vite@latest examples -- --template vanilla-ts
```

> **若已存在 examples 資料夾**：可先重新命名或刪除後再執行。

---

## 步驟 2：安裝依賴並啟動

```bash
cd examples
npm install
npm run dev
```

---
