# 範例 0：建立練習專案（可選）

## 目標

取得可在瀏覽器執行 TypeScript、並用 **Console** 與**頁面**觀察輸出的練習環境。

---

## 前置條件

- 已完成 [01 - 環境建置](../01-環境建置/主題.md) 與 [03 - TypeScript 配置](../03-TypeScript配置/主題.md) 的基礎認識

---

## 建議做法：使用各範例內建專案（推薦）

本單元已為 [範例 1](範例1-變數與基本型別.md)～[範例 5](範例5-聯合型別與型別窄化.md) 各準備**一個可複製的資料夾**，路徑在 **`08-TypeScript語法/examples/`** 下：

| 範例 | 資料夾 |
|------|--------|
| 範例 1 | `examples/01-basics/` |
| 範例 2 | `examples/02-functions/` |
| 範例 3 | `examples/03-interfaces/` |
| 範例 4 | `examples/04-arrays-generics/` |
| 範例 5 | `examples/05-unions/` |

每個資料夾內含 **`index.html`**、**`src/main.ts`**、**`src/style.css`**、該範例的 **`src/practice-….ts`** 與 Vite 設定。將資料夾複製到任意位置後：

```bash
cd 該範例資料夾
npm install
npm run dev
```

詳見 [examples 目錄說明](examples/README.md)。

---

## 替代做法：自行建立空白 Vite 專案

若你希望從零建立（例如自訂資料夾名稱），可在 **`08-TypeScript語法`** 下執行：

```bash
cd 08-TypeScript語法
npm create vite@latest my-ts-practice -- --template vanilla-ts
cd my-ts-practice
npm install
npm run dev
```

再自行新增練習用 `.ts` 並在 `main.ts` 以 `import` 載入；若要與講義一致，可對照各範例資料夾內的 `main.ts`（含將 `console.log` 同步到頁面的寫法）。
