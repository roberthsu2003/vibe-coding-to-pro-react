# 05 - React 核心概念

## 主題說明

本單元介紹 React 的重要概念，每個概念都有**可執行的互動範例**，學生可透過手動操作理解每個概念。

> **進階**：讀完本單元後，可至 [09 - React 單元（進階）](../09-React/主題.md) 學習 `useRef`、Context、自訂 Hook 等主題。

### 學習目標

- 理解 JSX 語法與大括號 `{}` 的用法
- 學會用 Props 傳遞參數
- 掌握 useState 管理狀態
- 認識 useEffect 處理副作用
- 學會事件處理與型別

### 學習流程

1. **範例 0**：一步一步建立 `examples` 專案框架
2. **範例 1～5**：依序實作各範例，完成範例 5 後即擁有完整的互動範例專案
