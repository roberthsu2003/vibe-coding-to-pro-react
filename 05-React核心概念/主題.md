# 05 - React 核心概念

## 主題說明

本單元介紹 React 的重要概念，每個概念都有**可執行的互動範例**，學生可透過手動操作理解每個概念。

### 學習目標

- 理解 JSX 語法與大括號 `{}` 的用法
- 學會用 Props 傳遞參數
- 掌握 useState 管理狀態
- 認識 useEffect 處理副作用
- 學會事件處理與型別

### 啟動互動範例

在開始各範例前，請先啟動範例專案：

```bash
cd 05-React核心概念/examples
npm install
npm run dev
```

瀏覽器開啟後，可透過上方按鈕切換不同範例，**手動操作**體驗每個概念。
