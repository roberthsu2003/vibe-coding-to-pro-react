# 範例 5：事件處理

## 目標

學會處理 `onClick`、`onChange` 等事件，並正確設定 TypeScript 型別。

---

## 步驟 1：建立輸入處理

```tsx
import { useState } from 'react'

function InputDemo() {
  const [text, setText] = useState('')

  const handleChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    setText(e.target.value)
  }

  return (
    <div>
      <input
        type="text"
        value={text}
        onChange={handleChange}
        placeholder="輸入一些文字..."
      />
      <p>即時顯示：{text || '(尚未輸入)'}</p>
    </div>
  )
}
```

**重點**：`e.target.value` 取得輸入內容，更新 `text` 狀態。

---

## 步驟 2：加入按鈕點擊

```tsx
const handleClick = () => {
  alert(`你輸入的是：${text}`)
}

return (
  <div>
    <input ... />
    <button onClick={handleClick}>顯示輸入內容</button>
    <p>即時顯示：{text || '(尚未輸入)'}</p>
  </div>
)
```

**重點**：`onClick` 接收函式，點擊時執行。

---

## 步驟 3：事件型別對照

| 事件 | 型別 |
|------|------|
| 輸入變更 | `React.ChangeEvent<HTMLInputElement>` |
| 點擊 | `React.MouseEvent<HTMLButtonElement>` |
