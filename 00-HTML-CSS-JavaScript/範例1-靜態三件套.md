# 範例：靜態三件套（計數器）

## 目的

用一個極小的頁面體驗：**改程式 → 存檔 → 手動重新整理瀏覽器** 的開發迴圈。

## 檔案位置

本範例程式在資料夾 [範例/靜態頁面](靜態頁面/) 內。

## 操作步驟

1. 用檔案總管或編輯器開啟 `靜態頁面/index.html`（雙擊以預設瀏覽器開啟即可）。
2. 點擊「+1」按鈕，確認數字會增加。
3. 修改 `靜態頁面/main.js`（例如把初始值改成 `10`），**存檔後**按瀏覽器重新整理，觀察變化。

## 對照思考

- 若要把邏輯拆成多個 `.js` 並用 `import`／`export` 組裝，在純檔案開啟情境下常會遇到路徑與 CORS 限制，通常要改成本機靜態伺服器。
- 沒有型別檢查，改名或傳錯參數要到執行時才容易發現。
