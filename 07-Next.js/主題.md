# 07 - Next.js

## 主題說明

本單元介紹 **Next.js**，由 Vercel 推出的 React 全端框架。對象為已學過 React 核心概念的初學者。

### 學習目標

- 理解 React 與 Next.js 的關係
- 認識與 React Router v7（Remix）的差異及適合的專案類型
- 學會建立 Next.js 專案
- 認識 App Router 與專案結構
- 掌握 Server Component 與資料載入
- 學會 Server Actions 處理表單
