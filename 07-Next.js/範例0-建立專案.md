# 範例 0：建立專案

## 目標

使用官方 CLI 建立 Next.js 專案，作為後續範例的基礎。

---

## 步驟 1：建立專案

在 `07-Next.js` 資料夾中開啟終端機，執行：

```bash
cd 07-Next.js
npx create-next-app@latest examples
```

> **若已存在 examples 資料夾**：可先重新命名為 `examples-backup`，或刪除後再執行上述指令。

---

## 步驟 2：選擇選項（若為互動模式）

若 CLI 詢問選項，建議依序選擇：

- **TypeScript**：Yes
- **ESLint**：Yes
- **Tailwind CSS**：Yes
- **`src/` directory**：Yes
- **App Router**：Yes
- **Turbopack**：No（可選）

或使用預設值（加上 `--yes`）：

```bash
npx create-next-app@latest examples --ts --tailwind --eslint --app --src-dir
```

---

## 步驟 3：進入專案並安裝依賴

若 CLI 未自動安裝，請執行：

```bash
cd examples
npm install
```

---

## 步驟 4：啟動開發伺服器

```bash
npm run dev
```

瀏覽器開啟 `http://localhost:3000` 後，應看到 Next.js 歡迎頁面。
