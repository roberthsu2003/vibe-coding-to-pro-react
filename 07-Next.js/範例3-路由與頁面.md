# 範例 3：路由與頁面

## 目標

學會新增路由與頁面，理解 App Router 的檔案式路由與 `<Link>` 元件。

---

## 前置條件

- 已完成 [範例 2：認識專案結構](範例2-認識專案結構.md)

---

## 步驟 1：檔案即路由

在 `src/app/` 下建立資料夾，並在資料夾內加入 `page.tsx`：

```
app/
├── page.tsx          → /
├── about/
│   └── page.tsx      → /about
└── products/
    └── page.tsx      → /products
```

---

## 步驟 2：新增 About 頁面

建立 `src/app/about/page.tsx`：

```tsx
export default function About() {
  return (
    <main className="p-8">
      <h1 className="text-2xl font-bold">關於我們</h1>
      <p>這是使用 Next.js 建立的頁面。</p>
    </main>
  );
}
```

---

## 步驟 3：使用 Link 導覽

使用 Next.js 的 `<Link>` 取代 `<a>`，可獲得客戶端導覽（不整頁重新載入）：

```tsx
import Link from "next/link";

<Link href="/about">關於我們</Link>
```

---

## 步驟 4：在 layout 加入導覽

修改 `src/app/layout.tsx`，在 `{children}` 前加入導覽列：

```tsx
<nav className="flex gap-4 p-4 border-b">
  <Link href="/">首頁</Link>
  <Link href="/about">關於</Link>
  <Link href="/products">商品</Link>
  <Link href="/contact">聯絡</Link>
</nav>
{children}
```

---

## 步驟 5：驗證

1. 儲存檔案
2. 開啟 `http://localhost:3000`
3. 點擊導覽連結，應可切換頁面

---
