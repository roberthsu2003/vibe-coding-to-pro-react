# Next.js 範例專案

本專案為 [07-Next.js](../主題.md) 章節的範例，使用 **Next.js** 建立。

## 學習方式

本 `examples` 資料夾為**最原始的基本 template**。請依序完成 [範例 0](../範例0-建立專案.md)～[範例 5](../範例5-Server-Actions表單處理.md)，依照教學**逐步修改**此專案：

| 範例 | 修改內容 |
|------|----------|
| 範例 0 | 建立專案（或使用本資料夾作為起點） |
| 範例 2 | 認識專案結構 |
| 範例 3 | 新增 About 頁面、在 layout.tsx 加入導覽 |
| 範例 4 | 新增 Products 頁面（Server Component 資料載入）、在 layout 加入商品連結 |
| 範例 5 | 新增 Contact 頁面（Server Actions）、在 layout 加入聯絡連結 |

---

這是使用 [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) 建立的 [Next.js](https://nextjs.org) 專案。

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
