# 範例 3：安裝套件

## 目標

學會使用 npm 安裝新套件。

---

## 步驟 1：安裝為正式依賴

```bash
npm install axios
```

**說明**：安裝 `axios`，並加入 `dependencies`。

- 用於執行時需要的套件（如 API 請求、UI 元件庫）

---

## 步驟 2：安裝為開發依賴

```bash
npm install -D eslint
```

**說明**：安裝 `eslint`，並加入 `devDependencies`。

- `-D` 或 `--save-dev` 表示開發依賴
- 用於開發工具（如 linter、測試框架）

---

## 步驟 3：確認安裝結果

開啟 `package.json`，檢查是否已加入對應欄位：

```json
{
  "dependencies": {
    "axios": "^1.6.0",
    ...
  },
  "devDependencies": {
    "eslint": "^8.0.0",
    ...
  }
}
```

---

## 步驟 4：認識 package-lock.json

安裝後會產生或更新 `package-lock.json`。

- 記錄每個套件的**精確版本**
- 確保團隊成員安裝到相同版本
- **不需手動編輯**，建議一併提交到 Git

---

## 完成

你已學會安裝套件的基本操作。
